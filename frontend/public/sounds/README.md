# Звуковые эффекты для игрового урока

Эта папка содержит звуковые эффекты для игрового интерфейса уроков.

## Необходимые файлы:

1. **correct.mp3** - Звук правильного ответа (веселый колокольчик)
2. **wrong.mp3** - Звук неправильного ответа (мягкий буззер)
3. **levelup.mp3** - Фанфары при завершении урока
4. **click.mp3** - Звук клика по кнопке
5. **castle-ambient.mp3** - Фоновая музыка для замка

## Где скачать бесплатные звуки:

### Вариант 1: Freesound.org
- Регистрируемся на https://freesound.org
- Ищем звуки с лицензией CC0 (бесплатно для коммерческого использования)
- Примеры поиска:
  - "success chime" для correct.mp3
  - "error beep" для wrong.mp3
  - "fanfare victory" для levelup.mp3
  - "button click" для click.mp3
  - "medieval ambient" для castle-ambient.mp3

### Вариант 2: Zapsplat.com
- https://www.zapsplat.com
- Бесплатная регистрация
- Огромная библиотека звуков
- Поиск по категориям: UI/Game Sounds

### Вариант 3: Pixabay Sound Effects
- https://pixabay.com/sound-effects/
- Полностью бесплатно
- Не требует атрибуции

## Технические требования:

- Формат: MP3
- Размер: до 1MB каждый файл
- Качество: 128-192 kbps (для оптимизации загрузки)
- Длительность:
  - correct.mp3: 0.5-1 сек
  - wrong.mp3: 0.5-1 сек
  - levelup.mp3: 2-3 сек
  - click.mp3: 0.1-0.3 сек
  - castle-ambient.mp3: 30-60 сек (зацикленный)

## Временная заглушка:

Пока звуки не добавлены, компонент будет работать без звука (Howler пропускает ошибки загрузки).

## После добавления звуков:

Просто скопируйте файлы в эту папку и перезапустите приложение.
