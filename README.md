## 🎯 О проекте

**MyTeens.Space** - интерактивная образовательная платформа для развития эмоционального интеллекта подростков. Помогает освоить навыки построения здоровых границ, управления эмоциями и построения отношений.

### ✨ Ключевые возможности

- 🛡️ **44 интерактивных урока** по 4 модулям
- ⚖️ **Колесо баланса** - оценка в 8 сферах жизни
- 👨‍👩‍👧 **Родительский кабинет** - контроль прогресса
- 🎨 **Анимации как в Duolingo** - увлекательное обучение
- 💜 **Катя Карпенко** - виртуальный психолог-наставник
- 📊 **Геймификация** - XP, уровни, стрики

## 🚀 Быстрый старт

### Требования
- Node.js 18+
- Python 3.10+
- MongoDB 4.4+

### Установка

```bash
# Клонировать репозиторий
git clone <repo-url>
cd app

# Установить зависимости frontend
cd frontend
yarn install

# Установить зависимости backend
cd ../backend
pip install -r requirements.txt

# Настроить переменные окружения
cp .env.example .env
```

### Запуск

```bash
# Backend (в отдельном терминале)
cd backend
uvicorn server:app --reload --port 8001

# Frontend (в отдельном терминале)
cd frontend
yarn dev
```

Приложение доступно: `http://localhost:3000`
